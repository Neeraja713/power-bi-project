# Power BI Dashboard Project

## Project Overview
This Power BI project analyzes business data to uncover insights using interactive dashboards and visualizations.

## Tools Used
- Power BI
- SQL / Excel (for data source)
- GitHub

## Dashboard Highlights
- KPI cards for key metrics
- Sales trends analysis
- Region-wise performance
- Category-wise insights
- Interactive slicers and filters

## Key Business Insights
- Identified top-performing regions
- Analyzed monthly sales trends
- Highlighted low-performing categories

## Files Included
- .pbix file (Power BI dashboard)
- Dataset used for analysis
- Dashboard screenshots for quick viewing

## Conclusion
This dashboard helps stakeholders make data-driven decisions through interactive and visual analytics.
